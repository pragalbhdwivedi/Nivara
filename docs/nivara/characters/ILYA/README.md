# Sergeant Ilya Rane

## Character Visual Sheet (5/8) — Sergeant Ilya Rane

### Face Vibe
- Practical, grounded, “I’ve seen too much but I’m still here.”
- Kind eyes with fatigue at the edges.
- Expression: steady focus; compassion without softness.

### Outfit Silhouette
- Medic-scout hybrid gear: light armor plates over layered cloth.
- Cross-body satchel, bandage rolls, small vials.
- High boots, wrapped forearms, scarf/hood for weather.
- Everything looks used, repaired, and dependable.

### Palette
- Slate gray, muted white, field green.
- Accent: medical red strip (tiny—on a patch or band wrap).

### Signature Prop
- Field satchel with visible supplies.
- Bone-handled scalpel or small utility knife.
- A stitched armband/patch (medic mark).
