# Lady Sairi of the Veil

## Character Visual Sheet (2/8) — Lady Sairi of the Veil

### Face Vibe
- Composed, intelligent, unreadable—diplomacy as a weapon.
- Eyes: steady, evaluating, never startled.
- Expression: polite softness with an edge of command.

### Outfit Silhouette
- High-collar ceremonial coat with a narrow waist and long flowing panels.
- Translucent silver veil (short, face-framing) or half-veil drape.
- Structured shoulders, minimal armor plates hidden under fabric (practical but elegant).
- Gloves always (she doesn’t touch the world bare-handed).

### Palette
- Frost silver, pearl white, charcoal gray.
- Accent: faint icy-blue thread embroidery (subtle, only visible up close).

### Signature Prop
- Oath-seal ring or thin signet (snowgem set).
- Scroll case or folded oath parchment strips sealed with wax.
