# Arch-Architect Lys Marrow

## Character Visual Sheet (6/8) — Arch-Architect Lys Marrow

### Face Vibe
- Beautifully controlled, emotionally unreadable.
- Eyes: analytical, measuring, like she’s calculating your weakness.
- Expression: calm superiority—never raises voice, never doubts herself.

### Outfit Silhouette
- Architect-commander coat: long, tailored, geometric seams.
- Subtle metal spine/shoulder plating integrated into fabric (law-as-armor).
- Clean gloves, structured collar, no loose cloth.
- Looks like a walking blueprint turned into couture.

### Palette
- Obsidian black, gunmetal gray, cold white.
- Accent: thin gold lines (like circuit traces or blueprint marks).

### Signature Prop
- Contract key (metal key-shaped artifact).
- Clause-scroll (folded, sealed document) or a thin slate tablet with glyphs.
- Optional: geometric pendant (Sanctum sigil).
