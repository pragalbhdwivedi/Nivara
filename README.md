## Website
The Astro site lives in `/site` and deploys to GitHub Pages from this repository.

Local development:
```bash
cd site
npm install
npm run dev
```

GitHub Pages deploys via Actions, with `BASE_PATH` set to the repository name (e.g., `/Nivara`).
